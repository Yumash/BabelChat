# Настройка

## Диалог настроек

Открывается через кнопку в оверлее или системный трей → Настройки.

### Вкладка "Основные"

| Параметр | Описание |
|----------|----------|
| Ключ DeepL API | Ваш API-ключ с проверкой и отображением квоты |
| Путь к WoW | Автоопределение или ручной выбор. Кнопка "Установить аддон" |
| Язык интерфейса | RU, EN, ES |
| Ваш язык | Сообщения на этом языке не переводятся |
| Целевой язык | На какой язык переводить |
| Каналы | Вкл/выкл: Группа, Рейд, Гильдия, Сказать/Крик, Шёпот, Подземелье |
| Не переводить свои | Ваши сообщения не переводятся (имя определяется автоматически) |

### Вкладка "Оверлей"

| Параметр | Описание |
|----------|----------|
| Прозрачность | 20-100% |
| Размер шрифта | 8-20pt |
| Перевод включён по умолчанию | Запускать с включённым переводом |
| Отладочная консоль | Показать окно отладки |

### Вкладка "Горячие клавиши"

| По умолчанию | Действие |
|-------------|----------|
| `Ctrl+Shift+T` | Вкл/выкл перевод |
| `Ctrl+Shift+C` | Перевести буфер обмена |

Кликни на поле и нажми нужную комбинацию для настройки.

## Настройки аддона в WoW

В WoW: ESC → Интерфейс → Аддоны → BabelChat

| Раздел | Параметры |
|--------|-----------|
| Общие | Вкл/выкл словарь, цвет перевода |
| Категории | 12 переключателей для категорий терминов |
| Каналы | Какие каналы чата переводить |
| Язык | Целевой язык (14 доступных) |
| Компаньон | Вкл/выкл буфер для приложения-компаньона |
| Режим | Только словарь / Только оверлей / Оба |

## Команды аддона

| Команда | Описание |
|---------|----------|
| `/babel` | Показать справку |
| `/babel config` | Открыть настройки WoW |
| `/babel on` / `off` | Вкл/выкл словарный перевод |
| `/babel test` | Тест с примером |
| `/babel companion` | Статус буфера компаньона |
| `/babel poll on` / `off` | Вкл/выкл fallback опрос ChatFrame |
| `/babel log on` / `off` | Вкл/выкл логирование чата |

## config.json

Все настройки хранятся в `config.json` (создаётся автоматически). Резервная копия `config.json.bak` создаётся перед каждым сохранением.
