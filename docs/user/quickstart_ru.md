# Быстрый старт

## Требования

- Windows 10/11
- World of Warcraft (Retail — The War Within / Midnight)
- Бесплатный [ключ DeepL API](https://www.deepl.com/pro-api) (500К символов/месяц, получение за 2 минуты)

## Шаг 1: Скачивание

Скачай `BabelChat.zip` из [GitHub Releases](https://github.com/Yumash/BabelChat/releases).

Распакуй в любое место (Рабочий стол, Загрузки — неважно).

## Шаг 2: Запуск от администратора

Правый клик на `BabelChat.exe` → **Запуск от имени администратора**.

> BabelChat нужны права администратора для чтения памяти процесса WoW. Это то же требование, что у WarcraftLogs и WeakAuras Companion.

## Шаг 3: Мастер настройки

При первом запуске мастер проведёт тебя через 5 шагов:

### 3.1 Приветствие
Выбери язык интерфейса (Английский, Русский или Испанский).

### 3.2 Ключ DeepL API
1. Нажми ссылку для перехода на страницу регистрации DeepL
2. Зарегистрируйся на **бесплатный** аккаунт (без кредитной карты для Free)
3. Перейди в [Аккаунт DeepL](https://www.deepl.com/account/summary) → API Keys
4. Скопируй ключ и вставь в BabelChat
5. Нажми "Проверить" для подтверждения

### 3.3 Путь к WoW
BabelChat попробует автоматически найти WoW. Если не найдёт — нажми "Обзор" и выбери папку `World of Warcraft`.

### 3.4 Языки
- **Твой язык** — язык, на котором ты говоришь (сообщения на этом языке не переводятся)
- **Целевой язык** — на какой язык переводить иностранные сообщения

### 3.5 Установка аддона
Нажми "Установить аддон" — BabelChat скопирует аддон в папку WoW. Или скопируй `addon/BabelChat/` вручную:
```
World of Warcraft/_retail_/Interface/AddOns/BabelChat/
```

## Шаг 4: Запуск WoW

1. Запусти World of Warcraft
2. На экране выбора персонажа нажми **Аддоны** и убедись что "BabelChat" включён
3. Зайди в игру
4. В чате появится: *"Добро пожаловать в BabelChat!"*

## Шаг 5: Играй

Зайди в группу и общайся. BabelChat будет:
- Показывать **оригинальное сообщение мгновенно** в оверлее
- Показывать **перевод через 0.5-2 секунды** под ним
- Частые фразы (gg, ty, brb) переводятся **мгновенно** без задержки

## Что ты увидишь

Оверлей появляется поверх WoW с тёмным полупрозрачным фоном:

- **Цвета каналов** совпадают с WoW (синий — Группа, оранжевый — Рейд, зелёный — Гильдия)
- **Оригинал** серым цветом
- **Перевод** золотым, со стрелкой: `→ переведённый текст`
- **Вкладки-фильтры** сверху для показа только нужных каналов

## Советы

- **Переключи перевод** нажатием `Ctrl+Shift+T` (настраивается)
- **Ответь на другом языке**: кликни в поле ответа, напиши сообщение, Enter → перевод скопирован в буфер → вставь в WoW через `Ctrl+V`
- **Сверни оверлей** кнопкой в заголовке
- **Двигай/изменяй размер** перетаскиванием заголовка или нижнего правого угла
- Оверлей **прозрачен для кликов** — клики проходят сквозь него в WoW
