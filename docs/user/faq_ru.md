# FAQ и решение проблем

## Общие вопросы

### Зачем BabelChat нужны права Администратора?
Для чтения памяти процесса WoW через `ReadProcessMemory`. Это Windows API, требующий повышенных привилегий.

### Это безопасно? Меня не забанят?
BabelChat только **читает** память — никогда не пишет, не инъектирует, не автоматизирует. Warden (античит WoW) не реагирует на чтение памяти.

### Почему не просто читать файл чат-лога?
Мы пробовали. WoW буферизирует `WoWChatLog.txt` с буфером ~4КБ и сбрасывает непредсказуемо — задержки от 1 до 5+ минут, сообщения приходят пачками в случайном порядке. Для переводчика в реальном времени это бесполезно. Наш аддон пишет в Lua-строку в памяти, компаньон читает её каждые 250мс — задержка менее секунды. Этот подход уникален: WeakAuras Companion читает SavedVariables с диска (нужен `/reload`), WarcraftLogs тейлит combat log (недоступен для чата).

### Почему перевод занимает 0.5-2 секунды?
Оригинальное сообщение появляется **мгновенно**. Задержка — это round-trip до серверов DeepL (текст → нейросеть → перевод обратно). Частые фразы (gg, ty, brb, hello) переводятся мгновенно из встроенного разговорника — без обращения к API.

### Сколько сообщений можно перевести бесплатно?
DeepL Free даёт 500 000 символов/месяц — примерно 10 000 сообщений. Для большинства игроков этого более чем достаточно. Платные планы DeepL дают безлимитный перевод.

## Проблемы с оверлеем

### Оверлей пустой / сообщения не появляются
1. Проверь что аддон BabelChat включён в WoW (выбор персонажа → Аддоны)
2. Напиши `/babel` в чате WoW — должна появиться справка
3. Проверь что в заголовке оверлея написано "WoW: Connected"
4. Убедись что BabelChat.exe запущен **от Администратора**

### Оверлей закрывает важные части экрана
- **Перетащи** за заголовок
- **Измени размер** за нижний правый угол
- **Сверни** кнопкой в заголовке
- Оверлей запоминает позицию и размер между сессиями

### Клики проходят сквозь оверлей
Так задумано — оверлей прозрачен для кликов, чтобы можно было играть. Для взаимодействия наведи на область заголовка.

## Проблемы с переводом

### Сообщения на моём языке переводятся
Проверь Настройки → Основные → "Ваш язык". BabelChat автоопределяет язык, но настройка "ваш язык" указывает какой пропускать.

### Некоторые короткие сообщения не переводятся
Сообщения типа "ok", "lol", "kk" в списке пропуска — они универсальны. Аббревиатуры "gg", "ty" переводятся через разговорник вместо DeepL.

### Плохое качество перевода игровых терминов
Встроенный словарь обрабатывает частые термины (dps, lfm, wts). Если DeepL неправильно переводит игровой жаргон — нейросеть не знает контекст WoW. BabelChat отправляет подсказку контекста в DeepL, но сложный жаргон может переводиться неидеально.

## Проблемы с подключением

### "WoW: Searching..." в оверлее
Memory reader сканирует память WoW в поисках буфера аддона. Первое подключение может занять 1-3 секунды, или после `/reload`. Если проблема сохраняется:
1. Убедись что аддон установлен и включён
2. Напиши `/babel companion` в WoW для проверки буфера
3. Перезапусти BabelChat.exe

### BabelChat не находит WoW
Убедись что WoW запущен до подключения. Приложение будет пытаться каждые 5 секунд.

## Проблемы с аддоном

### "/babel" не работает в WoW
1. Проверь что аддон включён: Выбор персонажа → Аддоны → BabelChat
2. Напиши `/reload` для перезагрузки аддонов
3. Проверь Lua ошибки: `/console scriptErrors 1`

### Я обновился с ChatTranslatorHelper
BabelChat автоматически мигрирует настройки. Просто установи BabelChat и удали старую папку ChatTranslatorHelper из AddOns.
